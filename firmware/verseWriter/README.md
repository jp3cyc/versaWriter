# プログラムの書き込み方法

このバーサライタはシリアル経由での書き込みが何故かできません．
そのため，ISP経由の書き込みをおこないます．

ISP経由の書き込みの方法を次に示します．

# インストール編

- avrdudeをインストールして，PATHを設定します．ArduinoIDEをインストールすると一緒に書き込まれているかも
- ArduinoIDEの設定を変更して，hexファイルを出力する設定に変更します．http://d.hatena.ne.jp/licheng/20130826/p1 この設定の時出力するファイル専用のディレクトリを作成してください．コンパイルのたびにフォルダ内をすべて削除するので，デスクトップに設定するとコンパイルのたびにデスクトップのファイルを全て削除してくれます．デスクトップ以下にディレクトリを作りましょう．

# 書き込み編
- コマンドプロンプトを起動
- hexファイルのあるディレクトリまでcdコマンドを使って移動
- derコマンドをつかって，.hexで終わるファイルを探す
- verseWriter.cpp.hexだったとします．そのときには次のコマンドを打ち込みます．（まだEnterはおさない）

avrdude -c avrisp -P COM9 -b 19200 -p m328p -U flash:w:verseWriter.cpp.hex

- ArduinoISPを書き込んだArduinoにISP変換シールドを乗せます．
- ArduinoISPをパソコンと接続した後，バーサライタの電源を入れて，Arduinoシールドとバーサライタを接続します
- コマンドプロンプトでEnterを押して書き込みます
